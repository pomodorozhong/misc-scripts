# pdf rotate

## Run

### Windows

```bash
python .\main.py
```

### OS X & Linux

```bash
python .\main.py
```

## Installation of virtual environment

### Windows

```bash
# setup basic environment
python -m venv .venv

# activate virtual environment
.venv/Scripts/Activate.ps1

# fetch dependency
pip install -r requirements.txt
```

### OS X & Linux

```bash
# setup basic environment
python3 -m venv .venv

# activate virtual environment
source .venv/bin/activate

# fetch dependency
pip3 install -r requirements.txt
```
